# AirconPi

## 概要 (Overview)

SlackBotに話しかけてエアコンを操作します。

・電源オン/オフ<br>
・設定温度アップ/ダウン<br>
・部屋の温湿度・気圧の取得<br>
・温度設定の予約<br>

が出来ます。
